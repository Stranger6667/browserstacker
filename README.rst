BrowserStacker
==============
Python wrapper for BrowserStack features.

|Build Status| |codecov.io|

Python support
--------------

BrowserStacker supports Python 2.6, 2.7, 3.2, 3.3, 3.4, 3.5, PyPy, PyPy3 and Jython.
